<h2 id="clone">Clone</h2>

<h4> Prerequisites</h4>

- Node >= 16.17.0 e npm >= 5.6 
- Package manager - NPM or YARN

<h4>Starting</h4>

```
git clone https://github.com/valmarath/fake-store
npm install
```

<h5>Start Client</h5>

```
  //in root
  npm run dev
```
